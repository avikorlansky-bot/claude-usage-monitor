# Screenshots

The main `README.md` references three images in this folder:

- `menubar.png` — the status-bar item itself (cropped to a wide strip showing `Claude 23%` next to your other menu-bar icons).
- `dropdown.png` — the menu open, showing the rows for 5-hour, weekly all-models, and any other limits.
- `floating.png` — the optional always-on-top floating window.

Drop those three files in here with those exact names and the README will pick them up automatically.

## How to capture them on macOS

- Press **⌘ + Shift + 4**, then **Space**, then click the menu bar / dropdown / floating window to capture just that window with a transparent shadow.
- Or **⌘ + Shift + 4** and drag a rectangle for a custom region.
- PNG is the preferred format. ~2x retina resolution looks best on GitHub.
